# WatchPusher: 微信消息推送助手 (毕设版)

本程序由 AI 自动生成，专为 HH-D101 手表开发。

## 🚀 如何在 2 分钟内获取 APK (无需安装 Android Studio)

### 第一步：代码托管
1. 登录您的 **GitHub** 账号，点击 **New repository**。
2. 将 `g:\app` 文件夹内的**所有内容**上传至该仓库。

### 第二步：云端打包
1. 点击仓库顶部的 **Actions** 标签。
2. 您会看到一个名为 **"Android Build APK"** 的任务正在自动运行（图标在转圈）。
3. 等待约 2 分钟，图标变绿：
   - 点击该任务进度计划。
   - 在页面底部找到 **Artifacts** 区域。
   - 点击 **WatchPusher-APK** 即可下载压缩包。

### 第三步：手机端操作
1. 解压下载的文件，将 `app-debug.apk` 发送到手机并安装。
2. 打开 App，点击 **1. GRANT NOTIFICATION ACCESS**，在系统列表里勾选 **Watch Notification Service**。
3. 回到 App，点击 **2. CONNECT TO WATCH**。
4. 现在可以请朋友给您发一条微信试试了！

---
> [!IMPORTANT]
> **演示技巧**：如果微信通知没有触发，请检查手机是否开启了“通知亮屏”或“悬浮通知”。
